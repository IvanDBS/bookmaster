# Shared

Общие типы, утилиты и конфигурации для проекта BookMaster.

## Структура

```
shared/
├── types/           # TypeScript типы для API
├── utils/           # Общие утилиты
├── constants/       # Константы
└── README.md
```

## Использование

### Backend (Rails)
- Импорт общих констант и утилит
- Валидация данных

### Frontend (Vue)
- TypeScript типы для API
- Общие компоненты
- Утилиты форматирования
